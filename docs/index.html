Mode d'Emploi

01

Prérequis

Installez Node.js sur votre ordinateur. C'est le moteur qui permet à Jarvis de fonctionner.

02

Le Cerveau

Obtenez votre clé API gratuite sur Google AI Studio. C'est ce qui donne son intelligence à Jarvis.

03

Configuration

Créez un fichier .env dans le dossier backend et collez votre clé : GEMINI_API_KEY=votre_cle.

04

Lancement

Lancez le backend et le client avec npm start. Appuyez sur Alt + J et dites "Bonjour Jarvis" !
